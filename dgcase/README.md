## 程式碼架構：
### dgcase1.ipynb和dgcase2ipynb是我們分別做的檔案，包含全部簡報和報告的內容

### dgcase1:
* import library
* 基本資料分析資料、畫圖
* 基本資料預測是否住院
* 預測和畫圖，分別用，merge 4種濃度、merge AST ALT和merge AST/ALT 
* 預測死亡病人與分析死亡病人數據

### dgcase2:
* 生死和診斷月份的關係圖
* 生死和發病月份的關係圖
* 住院與否和診斷月份的關係圖
* 住院與否和發病月份的關係圖
* 住院與否和性別、年紀的關係
* 住院與否和診斷月分、AST, APTT ,ALT, Platelet value之間的關係
* 個物質濃度的分佈
* 住院與否和AST/ALT的關係
